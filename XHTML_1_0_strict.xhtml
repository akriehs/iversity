"XHTML 1.0 Strict"-Elemente
Element-Tag Elementbeschreibung Inlineelement Blockelement Semantisches HTML
<a href="URL">...</a> und <a name="$"> Verweis (Hyperlink) und Anker für Verweise (Anchor)


<abbr>...<abbr/> Abkürzung (Abbreviation)


<acronym>...</acronym> Akronym (Acronym)


<address>...</address> Autoreninformation (Author information)


<area /> Verweis-sensitiver Bereich in einer Image-Map-Grafik (Client-side image map) Headertag

<b>...</b> Fettschrift (Bold text style)


<base href="URI" /> Basis-URI (Document base URI)


<bdo>...</bdo> Richtungsänderung bei bidirektionalem Text, sog. I18N: Bidi-Eingriff (I18N: Bidi override)


<big>...</big> Großschrift (Large text style)


<blockquote>...</blockquote> Zitat in Form eines eigenen Textabsatzes (Long quotation)


<body>...</body> Inhaltsbereich eines HTML-Dokuments oder Dokumentkörper (Document body) Dokumentstrukturtag

<br /> Erzwungener Zeilenumbruch (Forced line break)


<button>...</button> Schaltfläche in einem Formular (Push button)


<caption>...</caption> Tabellenüberschrift / -unterschrift (Table caption)


<cite>...</cite> Zitat innerhalb eines Absatzes (Citation)


<code>...</code> Quelltext-Auszug innerhalb eines Absatzes (Code fragment)


<col /> Tabellenspalte (Table column)


<colgroup>...</colgroup> Gruppe von Tabellenspalten (Table column group)


<dd>...</dd> Definition, Beschreibung, detaillierter Inhalt zu einem dt-Element (Definition description)


<del>...</del> gelöschter Inhalt bzw. Änderungsmarkierung (Deleted text)


<dfn>...</dfn> Definition innerhalb eines Absatzes (Instance definition)


<div>...</div> Generischer Container (Generic container)


<dl>...</dl> Liste mit dt- und dd-Elementen, für Definitionslisten, Dialoge usw. (Definition list)


<dt>...</dt> Definierter Ausdruck in einer Definitionsliste, Sprechername in einem Dialog usw. (Definition term)


<em>...</em> Betonung (Emphasis)


<fieldset>...</fieldset> Gruppe von Elementen in einem Formular, Reiter in einem Dialog (Form control group)


<form>...</form> Formular (Interactive form)


<h1>...</h1>; <h2>...</h2>; <h3>...</h3>; <h4>...</h4>; <h5>...</h5>; <h6>...</h6> Überschrift 1. bis 6. Ordnung (Heading)


<head>...</head> Kopfbereich eines HTML-Dokuments (Document head) Dokumentstrukturtag

<hr /> Horizontale Trennlinie (Horizontal rule)


<html>...</html> Dokumentelement von HTML-Dokumenten, schließt das gesamte Dokument ein (Document root) Dokumentstrukturtag

<i>...</i> Kursivschrift (Italic text style)


<img src="URL" /> Referenzierte und eingebettete Grafik (Embedded image)


<input>...</input> Formularelement, verwendbar für diverse Typen wie Eingabefelder, Schaltflächen, Auswahlfelder usw. (Form control)


<ins>...</ins> Eingefügter Text (Inserted text)


<kbd>...</kbd> Darstellung eines über Tastatur einzugebenden Textes im Text (Text to be entered)


<label>...</label> Beschriftungstext eines Formularfeldes (Form field label)


<legend>...</legend> Überschrift für eine fieldset-Gruppe von Elementen in einem Formular (Fieldset legend)


<li>...</li> Listenpunkt in einer ul-, ol- oder menu-Liste (List item)


<link /> Logische Beziehungen der aktuellen Webseite zu anderen Webseiten (Media-independent link) Headertag

<map>...</map> Bereich für verweis-sensitive Grafiken (Client-side image map)


<meta /> Meta-Angabe für nicht im Inhalt sichtbare Dokumentinformationen oder für Anweisungen an den Webserver (Generic meta-information)


<noscript>...</noscript> Bereich für Inhalte, die angezeigt werden, wenn JavaScript im Browser deaktiviert oder nicht verfügbar ist (Alternate container)


<object>...</object> Eingebettetes Objekt, geeignet für spezielle Multimedia-Objekte, Java-Applets usw. (Generic embedded object)


<ol>...</ol> Geordnete (z.B. nummerierte) Liste (Ordered list)


<optgroup>...</optgroup> Gruppe von Auswahlmöglichkeiten in einer Auswahlliste (Option group)


<option>...</option> Auswahlmöglichkeit in einer Auswahlliste (Selectable choice)


<p>...</p> Textabsatz (Paragraph)


<param>...</param> Parameter eines Objekts bei object-Element (Named property value)


<pre>...</pre> Vorformatierter Text (Plain text)


<q>...</q> Wörtliche Rede in einem Text (Short inline quotation)


<samp>...</samp> Beispieltext in einem dokumentierenden Text (Sample output)


<script>...</script> Scriptbereich für JavaScript und andere Scriptsprachen (Script statements)


<select>...</select> Auswahlliste in einem Formular (Option selector)


<small>...</small> Kleinschrift (Small text style)


<span>...</span> Allgemeiner Bereich innerhalb eines Absatzes (Generic container)


<strong>...</strong> Stark betonter Inhalt im Text (Strong emphasis)


<style>...</style> Bereich für Stylesheet-Definitionen, z.B. in CSS (Style info)


<sub>...</sub> Tiefgestellter Text (Subscript)


<sup>...</sup> Hochgestellter Text (Superscript)


<table>...</table> Tabelle (Table)


<tbody>...</tbody> Tabellenkörper mit den eigentlichen Daten (Table body)


<td>...</td> Tabellenzelle (Table data cell)


<textarea>...</textarea> Mehrzeiliges Textfeld (Multi-line text field)


<tfoot>...</tfoot> Fußbereich in einer Tabelle, z.B. für summarische Daten (Table footer)


<th>...</th> Tabellenkopfzelle (Table header cell)


<thead>...</thead> Kopfbereich einer Tabelle (Table header)


<title>...</title> Titel eines HTML-Dokuments (Document title)


<tr>...</tr> Tabellenzeile (Table row)


<tt>...</tt> Fernschreiber-Text (Teletype text style)


<ul>...</ul> Unsortierte Liste (Aufzählungsliste) (Unordered list)


<var>...</var> Variabler Name in einem dokumentierenden Text (Variable instance)
