Element-Tag Elementbeschreibung Inlineelement Blockelement Semantisches HTML
a Verweis (Hyperlink) und Anker für Verweise (Anchor)


abbr Abkürzung (Abbreviation)


acronym Akronym (Acronym)


address Autoreninformation (Author information)


area Verweis-sensitiver Bereich in einer Image-Map-Grafik (Client-side image map) Headertag

b Fettschrift (Bold text style)


base Basis-URI (Document base URI)


bdo Richtungsänderung bei bidirektionalem Text, sog. I18N: Bidi-Eingriff (I18N: Bidi override)


big Großschrift (Large text style)


blockquote Zitat in Form eines eigenen Textabsatzes (Long quotation)


body Inhaltsbereich eines HTML-Dokuments oder Dokumentkörper (Document body) Dokumentstrukturtag

br Erzwungener Zeilenumbruch (Forced line break)


button Schaltfläche in einem Formular (Push button)


caption Tabellenüberschrift / -unterschrift (Table caption)


cite Zitat innerhalb eines Absatzes (Citation)


code Quelltext-Auszug innerhalb eines Absatzes (Code fragment)


col Tabellenspalte (Table column)


colgroup Gruppe von Tabellenspalten (Table column group)


dd Definition, Beschreibung, detaillierter Inhalt zu einem dt-Element (Definition description)


del gelöschter Inhalt bzw. Änderungsmarkierung (Deleted text)


dfn Definition innerhalb eines Absatzes (Instance definition)


div Generischer Container (Generic container)


dl Liste mit dt- und dd-Elementen, für Definitionslisten, Dialoge usw. (Definition list)


dt Definierter Ausdruck in einer Definitionsliste, Sprechername in einem Dialog usw. (Definition term)


em Betonung (Emphasis)


fieldset Gruppe von Elementen in einem Formular, Reiter in einem Dialog (Form control group)


form Formular (Interactive form)


h1 bis h6 Überschrift 1. bis 6. Ordnung (Heading)


head Kopfbereich eines HTML-Dokuments (Document head) Dokumentstrukturtag

hr Horizontale Trennlinie (Horizontal rule)


html Dokumentelement von HTML-Dokumenten, schließt das gesamte Dokument ein (Document root) Dokumentstrukturtag

i Kursivschrift (Italic text style)


img Referenzierte und eingebettete Grafik (Embedded image)


input Formularelement, verwendbar für diverse Typen wie Eingabefelder, Schaltflächen, Auswahlfelder usw. (Form control)


ins Eingefügter Text (Inserted text)


kbd Darstellung eines über Tastatur einzugebenden Textes im Text (Text to be entered)


label Beschriftungstext eines Formularfeldes (Form field label)


legend Überschrift für eine fieldset-Gruppe von Elementen in einem Formular (Fieldset legend)


li Listenpunkt in einer ul-, ol- oder menu-Liste (List item)


link Logische Beziehungen der aktuellen Webseite zu anderen Webseiten (Media-independent link) Headertag

map Bereich für verweis-sensitive Grafiken (Client-side image map)


meta Meta-Angabe für nicht im Inhalt sichtbare Dokumentinformationen oder für Anweisungen an den Webserver (Generic meta-information)


noscript Bereich für Inhalte, die angezeigt werden, wenn JavaScript im Browser deaktiviert oder nicht verfügbar ist (Alternate container)


object Eingebettetes Objekt, geeignet für spezielle Multimedia-Objekte, Java-Applets usw. (Generic embedded object)


ol Geordnete (z.B. nummerierte) Liste (Ordered list)


optgroup Gruppe von Auswahlmöglichkeiten in einer Auswahlliste (Option group)


option Auswahlmöglichkeit in einer Auswahlliste (Selectable choice)


p Textabsatz (Paragraph)


param Parameter eines Objekts bei object-Element (Named property value)


pre Vorformatierter Text (Plain text)


q Wörtliche Rede in einem Text (Short inline quotation)


samp Beispieltext in einem dokumentierenden Text (Sample output)


script Scriptbereich für JavaScript und andere Scriptsprachen (Script statements)


select Auswahlliste in einem Formular (Option selector)


small Kleinschrift (Small text style)


span Allgemeiner Bereich innerhalb eines Absatzes (Generic container)


strong Stark betonter Inhalt im Text (Strong emphasis)


style Bereich für Stylesheet-Definitionen, z.B. in CSS (Style info)


sub Tiefgestellter Text (Subscript)


sup Hochgestellter Text (Superscript)


table Tabelle (Table)


tbody Tabellenkörper mit den eigentlichen Daten (Table body)


td Tabellenzelle (Table data cell)


textarea Mehrzeiliges Textfeld (Multi-line text field)


tfoot Fußbereich in einer Tabelle, z.B. für summarische Daten (Table footer)


th Tabellenkopfzelle (Table header cell)


thead Kopfbereich einer Tabelle (Table header)


title Titel eines HTML-Dokuments (Document title)


tr Tabellenzeile (Table row)


tt Fernschreiber-Text (Teletype text style)


ul Unsortierte Liste (Aufzählungsliste) (Unordered list)


var Variabler Name in einem dokumentierenden Text (Variable instance)
